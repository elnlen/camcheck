Contact

Bug reports, feature requests, or questions about how the camera tools behave — all welcome. I'll read through what you send and follow up with next steps or a fix plan where it makes sense. Replies can take a little while.

Send it through the form

Fill out the form below and submit it directly.

What to include in a bug report

  • Which page and feature it happened on (e.g. the 1080p step of the Resolution Test)
  • Your device and browser (e.g. MacBook Air / Safari 18)
  • The exact error message if one appeared (e.g. NotAllowedError in the status line)
  • A screenshot, if you can grab one

About the source code

This site is a static GitHub Pages site — there's no special server behind it. If you'd like to read the code, fix something yourself, or fork it, Issues and Pull Requests on the repo are welcome. The repo is here.
